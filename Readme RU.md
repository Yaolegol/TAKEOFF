# Старт проекта
> docker-compose up -d

После выполнения данной команды, приложение будет доступно по адресу:
>http://localhost:8000/

<strong>Порт</strong> можно задать в ```.env``` файле в корне проекта: ```DOCKER_COMPOSE__HOST_PORT```

## Доступные роуты
- главная страница
>http://localhost:8000/
