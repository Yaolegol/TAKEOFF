# Start project
> docker-compose up -d

After executing this command, the application will be available at:
>http://localhost:8000/

<strong>Port</strong> can be set in the ```.env``` file at the root of the project: ```DOCKER_COMPOSE__HOST_PORT```

### Available routes
- home page
>http://localhost:8000/
